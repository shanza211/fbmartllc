# FBMart LLC - One-Page eCommerce Website

A modern, responsive, and minimalist one-page eCommerce website built for the food products brand **FBMart LLC**.

## Tech Stack
- **Backend**: Node.js, Express.js (serving static files & API endpoints)
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla ES6), Bootstrap 5.3, Font Awesome

## Features
- **Modern White & Green Theme**: Clean design focused on freshness and food quality.
- **Dynamic Catalog**: Interactive product cards with star ratings, pricing, and quantities.
- **Fully-featured Cart**: Slide-out cart with subtotal calculation, quantity adjustments, and item persistent storage via `localStorage`.
- **Contact Form API**: Submits contact messages to the Node.js Express server.
- **Responsive Layout**: Works seamlessly on mobile, tablet, and desktop viewports.

## How to Run
1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   node server.js
   ```
4. Access the store at `http://localhost:3000`.
